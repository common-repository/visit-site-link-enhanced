== Description ==


The Wordpress PlugIn "Visit Site" Link Enhanced is a dead simple WP plugin which allows :

   1. to add a target ="_blank" attribute to the "Visit Site" link on the admin header,
   2. to add another optional link next to the "Visit Site" link



== Installation ==

1. Upload `the xvp_vsle folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings on the Admin Panel and set it up
